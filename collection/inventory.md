# NOT RED — Current Collection

> **Status:** STUB. Populate this file by dropping photos and notes about existing pieces into `reference/collection/`, then running the `notred-brand-steward` agent to ingest and draft the inventory. The user then reviews and edits.

Each piece should be documented as:

## <Piece name>
- **Type:** (ring, pendant, earrings, etc.)
- **Materials:** metals, stones, settings
- **Motifs:** which symbols are present
- **Status:** in collection / sold / one-of-one / open commission
- **Notes:** anything else worth remembering
